# Mobile Legends Match Tracker (Python)

A lightweight console-based application designed to help **Mobile Legends: Bang Bang** players track their performance across multiple matches. This script calculates KDA, tracks win rates, and assigns performance tags based on match outcomes.

## 🚀 Features

- **Player Profiling:** Input your IGN and current rank.
- **Hero Roster:** Uses a 2D list to manage a list of heroes and their respective roles.
- **Dynamic Match Entry:** Input stats (Kills, Deaths, Assists) and match results.
- **KDA Calculation:** Automatically calculates KDA (with zero-death protection).
- **Performance Tagging:** Assigns fun tags like "DOMINATION!" or "Carried Hard" based on stats.
- **Match Log Summary:** Generates a formatted summary table including:
  - Match history
  - Win/Loss count
  - Overall Win Rate (%)
  - Best Match highlights
